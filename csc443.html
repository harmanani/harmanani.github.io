CSC443

Web Programming

  • Course Description

    This course introduces advanced concepts in Web Programming and focuses on the development of dynamic web pages that incorporate both client-side and server-side programming.   Topics include web scripting using JavaScript, VBScript, and PHP; Java Beans, server side components such as CGI, ASP, and PHP, installing and configuring web servers.  The course also discusses accessing databases through web application.  Hands-on experience is required.

  • Course Learning Outcomes

    Students shall be able to demonstrate:

    1. Ability to effectively program using at least one current web programming language;
    2. Ability to use databases with web applications;
    3. Ability to develop server and client side scripts;
    4. Ability to work effectively in a team in order to successfully complete a substantial project by integrating various web technologies.

  • Project

    Your project should include at least four of the following grouped items with the first item being compulsory:

    1. HTML, CSS, secure forms, JavaScript, AJAX, JSON, cookies, and sessions;
    2. A Client-Side Framework such as jQuery or Angular;
    3. Node.JS;
    4. A non-structured database such as MongoDB or NoSQL;
    5. PHP and MySQL;
    6. A Web Service - use an external web service, mashed up with your own application to create something even better.

    Your report should document the use of these technologies!

     

     Your project will be graded based on the following:

    1. Innovation and novelty;
    2. Layout and look;
    3. Software design;
    4. Project planning and management;
    5. Oral presentation;
    6. Successfully completing the demo.

     

    Although you will be working indecently on the project, you will have five milestones that you have to keep in mind:

    1. September 23: Proposal submission including the name of the partner and the technology to be used.
    2. October 23: Project Specification and Design using UML.  This should include the database and the wireframe;
    3. November 10: The programming and testing of all scripts and modules;
    4. November 23: The programming and testing of the visual design;
    5. December 7-9: Projects presentations and reports.
