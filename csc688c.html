CSC 688C

Data Science and Visualization

  • Course Description

    This course covers the essential and practical skills necessary to communicate information about data clearly and effectively through graphical means based on principles from graphic design, visual art, perceptual psychology, and cognitive science.  The course introduces the value of visualization, the principles and techniques of data visualization, and specific techniques in information and scientific visualization.  The course also focuses on big data organization and mining for decision support, and how to best leverage visualization methods.  The assignments will involve the use of Tableau, web programming using D3, statistics, and the ability to manipulate data sets with code.  Prerequisite: Basic working knowledge of, or willingness to learn, graphics/visualization tools (D3, HTML5, OpenGL, and/or Google Tools) and data analysis tools (Tableau, R, Excel, Matlab) will be useful.

  • Course Learning Outcomes

    Students shall be able to demonstrate:

    1. The ability to understand the key techniques and theory used in visualization, including data preperation, data models, graphical perception and techniques for visual encoding and interaction.
    2. An exposure to a number of common data domains and corresponding analysis tasks, including multivariate data, networks, text and cartography.
    3. The ability to use tools in order to build and evaluate visualization systems.
    4. The ability to read, critique, and discuss research papers from the visualization literature.

  • Project and Grading

    Students will be required to complete a final visualization project near the end of the semester.  Students will select a data set and visualization technique, develop a prototype, and rework the prototype based on peer evaluations. Students will demonstrate their final projects during finals week.  There will be also several homework assignments, both written and programming, assigned on a bi-weekly basis. This will include evaluating and reworking existing visualizations, using existing tools to design visualizations, and prototyping custom visualizations in Processing or D3.

    1. Assignment 1: Visualization Design 10%
    2. Assignment 2: Exploratory Data Analysis 15%
    3. Assignment 3: Interactive Visualization 25%
    4. Midterm Presentation: 10%
    5. Final Project 40%

  • Data Sets

    1. Data.gov: U.S. government data.
    2. Data.gov.uk: U.K. government data.
    3. American FactFinder: Data resources from the U.S. Census Bureau.
    4. Center for Disease Control and Prevention: Website for public-use data files from the CDC.
    5. UCI Machine Learning Repository: Repository of data from UCI. Intended for machine learning, but often used for visualization examples.
    6. Stanford Large Network Dataset Collection: Network data sets, including social networks, citation networks, and more.
    7. Visual Analytics Benchmark Repository: Contains datasets and tasks from previous VAST and InfoVis Challenges/Contests.
    8. Social Explorer Reports: Historical and modern data from a variety of sources including the U.S. Census, American Community Survey, and ASARB.
    9. RealClimate: Various climate-related data sources.
    10. Project Gutenberg: Resource for free eBooks and texts (useful for text visualization examples).
    11. Datamob: Aims to show how public data sources can be used, and lists over 200 data sources.
    12. Click Dataset: A large dataset of HTTP requests from Indiana University.

  • Resources

    This course teaches the essential and practical skills necessary to communicate information about data clearly and effectively through graphical means. Rendering data clearly and effectively with appropriate visual analytics reduces the time required to achieve understanding and helps in managing the every growing amount of available digital data. Students will learn to use various software tools including Tableau, D3, Google Charts, Google Fusion Tables.
